# Club Dynasty

Et sesongbasert klubbmanagement-spill bygget med React, TypeScript og Vite.

## Spillinnhold

- 24 spillere i starttroppen
- 10 plasser i startlaget og automatisk «velg beste lag»
- Kontrakter, lønn, lønnstak, skader og spillerutvikling
- Spillermarked med ledige spillere adskilt fra egen spillerstall
- 8-klubbers liga med 14 serierunder
- Topp 4-sluttspill med semifinale og finale
- Autospill som stopper automatisk ved sluttspill og offseason
- Fans, omdømme, styretillit og styremål
- Tre sponsorkategorier og automatisk sponsorfornyelse som valgfri innstilling
- Arena, treningssenter, akademi og kommersiell avdeling
- Draft/akademispiller hver offseason
- Lokal automatisk lagring i nettleseren

## Kjør lokalt

```bash
npm install
npm run dev
```

## Bygg

```bash
npm run build
```
